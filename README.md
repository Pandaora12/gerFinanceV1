>> Para utilizar o app, é necessário instalar as dependências conforme abaixo:

> Instalação de Dependências: Execute os seguintes comandos no terminal na pasta do projeto...

> Instale o Electron: 
- npm install electron

>Instale o SDK do Supabase: 
- npm install @supabase/supabase-js

> Instale o Nodemailer para envio de emails:
- npm install nodemailer


>> Para iniciar o app, necessário executar o CMD abaixo:
- npm start

NO TERMINAL IRÁ APARECER O NOME DO PROJETO E O ELECTRON .

> Para conectar ao BD, necessário seguir o seguinte caminho:
/ cd gerFinanceV1
/ cd backend
/ cd services

node api.js

>> Necessário conectar ao banco antes de executar o app para sincronização dos dados.
